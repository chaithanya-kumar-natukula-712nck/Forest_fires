# Forest Fire Prediction System

## Problem Statement
Predict the likelihood of forest fire based on meteorological data.

## Dataset
Algerian Forest Fires Dataset

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML, Bootstrap

## ML Models Used
- Linear Regression
- Ridge
- Lasso
- ElasticNet

## How to Run
1. Clone repo
2. pip install -r requirements.txt
3. python application.py
